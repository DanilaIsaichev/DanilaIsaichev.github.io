Настройка линтера eslint для Visual Studio Code:

  1. Установка eslint
npm install -D eslint
  1. Установка плагинов eslint
npm install -D eslint-config-airbnb-base eslint-plugin-import
  1. Создание конфигурационного файла .eslintrc.yml
extends:
 - 'airbnb-base'
env:
 node: true
 browser: true
  1. Установка плагина ESLint
